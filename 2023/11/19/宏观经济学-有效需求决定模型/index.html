宏观经济学-有效需求决定模型

宏观经济学——Chapter2宏观经济模型

第四章 有效需求决定模型

有效需求与充分就业

有效需求： 总供给价格与总需求价格达到均衡时的社会产出总量和需求总量。

总供给价格： 一个社会全体厂商销售其产品时期望得到的最低价格总额。

总需求价格： 指一个社会全体居民购买消费品时愿意支付的最高价格总额。

如图，横轴看作商品的量，即社会的总产出，纵轴为整个社会商品的价格。AD：总需求，AS：总供给。

凯恩斯提到的有效需求就是当总需求和总供给达到均衡时，即图中的E点，所对应的价格水平和需求水平。

为什么AS曲线的斜率大于AD曲线的斜率 假设AS曲线的斜率小于AD曲线的斜率，即随着社会总产出规模的不断增大，总需求增长的速度大于 总产出增长的速度，即总产出越来越满足不了总需求，会刺激总产出无限扩大。这与社会现实显然是不符的。

决定一个社会就业水平的，就是E点对应的价格和需求水平。

因为E点对应的$ N_0$ 就是当前均衡的商品规模，则对应一个确定的就业量。若交点落在E点右边，如$N_2$，AS曲线高于AD，即所有厂商愿意接受的最低价格比所有居民愿意接受的最高价格要高，此时商品供过于求。厂商生产的产品有剩余，下一个生产周期内厂商一定会减少生产，则全社会的生产规模缩减，最后趋于$N_0$；反过来说，$N_1$点，AS曲线低于AD，即商品供不应求，厂商生产的产品不足，会在下一周期扩大生产，则全社会的生产规模扩大，同样趋于$N_0$

为什么AS曲线斜率大于AD？？（思考题）

充分就业： 指一个社会不存在非自愿失业时的就业状态。

自愿失业： 具有劳动能力的人出于种种原因自愿放弃工作而形成的失业。

摩擦性失业： 就业状态暂时中断而形成的失业。比如对当前工作不满意，为了找到更好的工作而暂时放弃当前工作的时期。

非自愿性失业： 有劳动能力且愿意接受现行工资水平或工作条件但仍然找不到工作而形成的失业。非自愿失业是由于经济衰退和萧条而造成的，因而也称周期性失业。

有效需求与充分就业的关系

$N_0$ = 充分就业（充分就业均衡）

$N_0 $> 充分就业（通货膨胀条件下的就业状态）

$N_0 $< 充分就业（经济衰退条件下的就业状态）

有效需求的决定模型

1.两部门经济的有效需求决定模型

$$
\begin{cases}
总供给Y = C + S\
总需求Y = C + I\
总需求 = 总供给I = S
\end{cases}
$$

如果居民在消费中并没有把所有的收入都用于消费，那么该循环更新为下图

如果居民将10%的收入用于储蓄，则下一轮生产周期，只有900亿被用来消费，即企业营收为900亿，如此类推，整个宏观经济是趋于收缩的。而当居民把储蓄存到银行，银行把收取的存款借款给企业，则能保持企业生产总规模至少维持1000亿不变。要维持原有的规模，则至少要保证银行吸收的储蓄和投资规模相等。

总结两部门经济的有效需求决定模型如下：

总供给 Y = C+S

总需求 Y = C+I

由此得到I = S，即投资等于储蓄，此时经济可以维持现有规模。如果I < S，即总需求小于总供给，则经济增长收缩；如果I > S，即总需求大于总供给，则经济增长扩张。

理论上，只要不断扩大投资规模就可以不断扩大经济总规模，但实际上经济生产是建立在可利用的资源之上的，如果资源支撑不了经济的发展，即经济增长超过其可利用资源的制约的话，经济过热、泡沫、通货膨胀等问题就会出现。

三部门经济的有效需求决定模型：

（pic）

此时

总供给 Y = C + S + T

总需求 Y = C + I + G(政府购买)

得到 I + G = S + T, I = S + T - G

T表示政府财政收入，G表示政府财政支出，则把$T-G$看作整体。

若I < S + (T-G)，总需求小于总供给，经济规模缩小。要维持经济规模保持不变，在投资和储蓄都确定的情况下，可以通过政府的财政活动，即扩大政府购买（财政赤字）。

若I > S + (T-G)，总需求大于总供给，经济规模扩大。此时政府采取的措施与上述情况相反。

由此可见，三部门经济相对于两部门经济，多了一个政府的调控过程。即尽量使经济活动中损失的经济回归到社会经济体中。

四部门经济的有效需求决定模型：

总供给 Y = C+S+T+M

总需求 Y = C+I+G+X

I = S + (T-G) + (M-X)，M-X称为净出口

同上所述，如果I < S，可以从T-G和M-X两个角度来调节经济规模

有效需求变动及其规律

$ Y = C+I$

$C = a+bY$

则$Y = (a+I) / (1-b)$

例1：已知消费函数C = 1000 + 0.8Y，投资I = 600亿，求总产出水平Y. Y = 8000亿

例2：假设上题中自发性消费从1000到900，I不变，求此时总产出水平Y. Y = 7500亿

由此可见，消费减少了100亿，但是总产出水平减少了500亿！

例3：假设消费函数仍为C = 1000 + 0.8Y，投资规模由600亿扩大至650亿，求总产出水平Y.    Y = 8250亿，$\Delta Y$ = 250亿. 即投资增加了50亿，总产出规模增加了250亿！

这里就引出消费乘数和投资乘数

宏观经济学中的“破窗理论”    宏观经济学中的”破窗理论”：一个男孩打破了窗玻璃，他妈妈拿钱去买了新玻璃，玻璃店老板雇佣新工人，工人拿钱去买汉堡......如此循环下去，原先一个窗玻璃的消费引发了后续一系列经济活动，并且其总和是远大于一个窗玻璃的价值。

这里就引发一个节约悖论，宏观经济的角度，消费带动经济发展，反而是节约阻碍了消费。而实际上现实生活中，低收入人群需要钱来消费但是没有钱，高收入人群的钱相对他的需求过多的，所以这里就产生了一部分需求不被满足，以及一部分钱闲置的浪费。所以政府的转移支付就尤为重要。

凯恩斯理论中有效需求的不足及其原因是边际消费倾向递减规律；投资的边际效率递减规律；灵活偏好和流动性陷阱。

消费乘数、投资乘数与政府购买乘数

政府对有效需求的调节：

$Y = C+I+G$

考虑财政转移支付g和政府税收T，则有国民可支配收入：

$ Yd = Y+g-T$

消费函数为：

$C = a+b(Y+g-T)$

整理后得到：

$Y = \dfrac{a+I+G+bg-bT}{1-b}$

$\Delta Y = \Delta a*\dfrac{1}{1-b}$，这里的$\dfrac{1}{1-b}$称为消费乘数

同理，$I$和$G$的变动对$Y$也有相同的影响效果，则$\dfrac{1}{1-b}$也可以用来表示投资乘数和政府购买乘数

现在考虑转移支付和税收：

$\Delta Y = \Delta g*\dfrac{b}{1-b}$，即转移支付乘数

$\Delta Y = \Delta T*\dfrac{-b}{1-b}$，即政府税收乘数

两种税收形式下的：

$Yd = Y + g - T - rY$，其中T称为固定税，rY称为比例税

相应的消费函数为：

$ C = a+b(Y+g-T-rY)$

整理可得$Y = \dfrac{a+I+G+bg-bT}{1-b+br}$，则各项乘数改写为

KC(消费乘数)=KI(投资乘数)=KG(政府购买乘数) = $\dfrac{1-r}{1-b}$

Kg(转移支付乘数) = $\dfrac{(1-r)b}{1-b}$

kT(政府税收乘数) = $\dfrac{-(1-r)b}{1-b}$
